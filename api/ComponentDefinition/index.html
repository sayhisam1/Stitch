ComponentDefinition

A ComponentDefinition is a table that defines a Component.

All ComponentDefinitions must be registered to a World before they can be used.

Example usage:

local positionComponent = {
	name = "position",
	defaults = {
		x = 0,
		y = 0,
		z = 0,
	},
}
world:registerComponent(positionComponent)
